# Supply Chain Transportation Optimization

Optimization project developed to minimize international transportation costs using real supply chain data.

## Features
- Linear Programming Model
- Supply & Demand Constraints
- Capacity Constraints
- Cost Minimization
- Python (PuLP) Implementation
- Julia (JuMP & GLPK) Implementation

## Dataset
SCMS Delivery History Dataset (Kaggle)

## Tools
Python, Julia, PuLP, JuMP, GLPK
